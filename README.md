# pico-sdk-tiny
Project configured for using Raspberry Pi's [Pico SDK](https://github.com/raspberrypi/pico-sdk) with [tiny](https://github.com/ryanplusplus/tiny).

## Usage
### Build
```shell
make
```

### Clean
```shell
make clean
```

### Upload
```shell
make upload
```

### Test
```shell
make test
```
